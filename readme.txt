# Netflix India Clone

This is my first Frontend Project built using HTML and CSS. 
Project focuses on standalone designs with some transitions that matches with current(2026) Netflix India design.
In project, i have designed a Netflix India clone using HTML and CSS.

#Live Demo
https://viraj-9.github.io/netflix-india-clone/
Ignore the warning showing to see the project(clone). It is showing that warning because i have used image links instead of the actual images on my local systems.
It is stand alone clone project it does not have backend so no need to worry about that warning.

#Built With
	- HTML5
	- CSS3 (Flexbox, Media Queries, Transitions)

#Features
	- Responsive design (Mobile, Tablet, Desktop)
	- Trending Now scrollable card section with number overlays
	- Hover animations and transitions
	- Curved section separator
	- FAQ section
	- Multi-column footer

#What I Learned
	- Flexbox layout
	- CSS transitions and hover effects
	- Responsive design with media queries
	- Z-index and positioning
	- CSS gradients and overlays
